# PullerBear Demo Scenario

This file exists to support a short product demo.

## Demo talking points

- PullerBear checks incoming changes before you pull.
- The extension summarizes remote commits in plain language.
- The chat lets you ask quick follow-up questions.

## Suggested demo flow

1. Open the PullerBear sidebar in VS Code.
2. Trigger a refresh on a branch that is behind the tracked branch.
3. Read the generated summary card.
4. Ask one short question in the chat box.
